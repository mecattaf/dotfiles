### Ponos Keybinds

|    Keybinds    |                Info               |
| -----          | -----                             |
| Win + drag window | move window around while holding Win key |
| Win + h/j/k/l | move focused floating window left/down/up/right |
| Win + left/down/up/right | move focused floating window left/down/up/right |
| Win + Shift + h/j/k/l | move focus around left/down/up/right |
| Win + Shift + down/up| shrink/grow focused floating window vertically |
| Win + Shift + left/right | shrink/grow focused floating window horizontally |
| Win + Shift + q            | kill focused window      |
| Win + Shift + Enter | launch terminal on base system |
| Win + Shift + r | reload sway |
| Win + i    | toggle waybar |
| Win + d     | start fzf dmenu launcher |
| Win + v    | show clipboard history |
| Win + Shift + v | show emoji selection menu |
| Win + s | take fullscreen screenshot and save in pictures |
| Win + Shift + s | screenshot portion of screen and save in pictures |
| Win + o | increase window opacity by 10% |
| Win + Shift + o | increase window transparency by 10% |
| Win + e | launch file explorer (nautilus) |
| Win + Tab | toggle focused window floating/tiling |
| Win + Shift + Tab | toggle focus between floating and tiling windows |
| Win + m | fzf menu to change workspace |
| Win + c | manually select other window to focus |
| Win + t | move focused window to other workspaces |
| Win + r | rename focused window |
| Win + n | name and create new workspace |
| Win + Backspace | next time you launch a program it spawns below corrently focused floating window |
| Win + a | focus all windows (tiled mode only) |
| Win + Shift + a | make focused window sticky (appears on all workspaces) |
| Win + f | make focused window fullscreen |
| Win + Shift + f | resize focused window to 900x500 |
| Win + g | resize focused window to 900x500 |
| Win + Shift + g | resize focused window to 900x500 |
| Win + comma | dismiss all notifications |
| Alt + Ctrl + Del | launch power menu |
