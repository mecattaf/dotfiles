# Impeccable — design fluency for AI harnesses

<https://impeccable.style>

A prompt-library (v2.1.1, original prompt-library era) of design skills:
`adapt`, `animate`, `audit`, `bolder`, `clarify`, `colorize`, `critique`,
`delight`, `distill`, `harden`, `impeccable`, `layout`, `optimize`,
`overdrive`, `polish`, `quieter`, `shape`, `typeset`.

## Why only this note

The pack ships one copy of every skill per harness (`claude/`, `cursor/`,
`codex/`, `gemini/`, `agents/`, …). Vendored under a scanned skills root, those
folders collided 11-way on every skill name and one skill (`harden`) carried
frontmatter that broke strict YAML parsers. Rather than keep a live copy here,
this is a reference pointer only.

To use it, fetch the current pack from <https://impeccable.style> and install
the `claude/` set into a project's `.claude/skills/`.
