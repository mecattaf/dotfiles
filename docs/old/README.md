# Old documentation

This directory is a lossless holding area for the documentation tree that
existed before the 2026-07-22 local-AI documentation reset. Its hierarchy is
preserved so the material can be mined before it is archived permanently.

Nothing here is normative. In particular, old Podman/toolbox commands, direct
backend ports, host roles, storage estimates, and scheduling language may
predate the NixOS transition or the current Tally and llama-swap architecture.
Use [`../local-ai/`](../local-ai/) and the live Nix modules for current truth.
