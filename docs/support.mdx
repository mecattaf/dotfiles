# Support

## Documentation

Start by exploring our documentation:

- [Getting Started](/docs/getting-started) - Installation and setup guide
- [DankMaterialShell](/docs/dankmaterialshell/overview) - Shell configuration and customization
- [dgop](/docs/dgop) - System monitoring tools
- [dsearch](/docs/danksearch) - Filesystem indexer and searcher
- [dankinstall](/docs/dankinstall) - Installer and management

## Community Support

### GitHub

Report bugs, request features, or browse existing issues:

- [DankMaterialShell Issues](https://github.com/AvengeMedia/DankMaterialShell/issues)
- [dgop Issues](https://github.com/AvengeMedia/dgop/issues)
- [dsearch Issues](https://github.com/AvengeMedia/danksearch/issues)
- [DankLinux Issues](https://github.com/AvengeMedia/DankLinux/issues)

### Discord

`DankMaterialShell` has a subsection on the niri discord server, keep relevant discussions in the relevant `#dms-*` channels.

**Discord Server:** [niri discord](https://discord.gg/ppWTpKmPgT)

### Bug Reports

When reporting bugs, please include:

- Your Linux distribution and version
- Dank Linux component versions
- Steps to reproduce the issue
- Expected vs. actual behavior
- Relevant logs or error messages
